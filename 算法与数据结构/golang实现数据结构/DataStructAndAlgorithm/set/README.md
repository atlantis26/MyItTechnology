集合：

    1、集合不包含重复元素，具有去重功能，这与二分搜索树的不重复元素节点吻合，所以可以使用二分搜索树来实现集合的功能；
    2、集合支持的方法： 
        add(e)：添加元素，不能添加重复的元素
        remove(e)： 移除元素
        contains(e): 判断是否存在元素e
        getSize(): 获取集合元素个数大小
        isEmpty(): 判断集合是否为空
        
    3、增删查等操作时间复杂度分析：
        1）以链表作为底层数据结构实现的集合LinkedSet,是O(n)的时间复杂度；
        2）以二分搜索树作为底层数据结构实现的集合BstSet, 是O(logn)的时间复杂度，相比链表实现，性能大大的提升；
     
     集合典型的应用： 客户统计、词汇量统计等需要去重的业务需求
     
    4、有序集合和无需集合
        在二分搜索树实现的集合是一个有序集合，基于搜索树实现的集合是有序的；
        基于链表实现的集合是无需集合；
        还有基于哈希表实现的集合是无需的集合；
     
     5、多重集合，集合中的元素可以重复：
        基于二分搜索树的集合，修改bst可以保存相同数据后，可以包装构成这种多重集合
